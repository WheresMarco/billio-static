Hjälp

På denna sida har vi samlat några vanliga frågor. Om detta inte hjälper er så kan ni alltid kontakta oss på [email protected].

På vilka sätt kan man betala?

Vi stödjer de mest populära bank- och kreditkorten. Det inkluderar Visa, Mastercard, och American Express. Tänk på att ert bank- eller kreditkort måste vara öppet för internetköp för att det ska fungera. Om ni inte kan betala med betala med bank- eller kreditkort hänvisar vi att kontakta arrangören för evenemanget.

Inom kort kommer det dessutom att vara möjligt att betala med Swish. Kontakta oss för att veta mer.

Är det säkert att betala med kort hos Billio?

Ja, det är helt säkert att betala med bank- och kreditkort hos Billio. Alla uppkopplingar mellan vår plattform och betalningssystem är krypterade. Det betyder att dina uppgifter aldrig skickas i klartext. Vi lagrar dessutom inga betaluppgifter hos oss. Ni kan läsa mer om hur vi arbetar med personuppgifter i vår Integritetspolicy.

Varför behöver ni veta mitt personnummer för kårrabatten?

Ditt personnummer hanteras enbart när ni vill använda er kårrabatt (endast tillgängligt vid Linköpings Universitet). Anledningen till detta är en begränsning hos systemen som hanterar medlemskap hos kårerna. Vi gör vårt yttersta att skydda era personuppgifter och ert personummer används inte på något annat än att verifiera ert medlemskap. Ni kan läsa mer om hur vi arbetar med personuppgifter i vår Integritetspolicy.

Jag har inte fått min biljett

Ibland kan det ta några minuter innan den kommer fram. Alternativt kan biljetten hamnat i skräpposten. Om du har kvar koden från bekräftelsesidan är den också gilltig som en biljett. Ni kan även kontakta oss för att få biljetten på nytt.

Jag kan inte gå på evenemanget längre, kan jag få pengarna tillbaka?

De allra flesta biljetter kan tyvärr inte återbetalas. Du kan alltid prova att kontakta arrangören direkt, för att se om de kan tänka sig att godkänna ditt återköp.

Kan jag sälja min biljett vidare?

Ja, det kan du. Tänk på att informera den nya biljettinnehavaren att uppge ditt namn vid incheckning då vi har ingen information om vem som köpt den nya biljetten. Om ni vill ändra innehavaren av biljetten kan ni kontakta oss så hjälper vi till med det.

Inom kort kommer det dessutom att vara möjligt att sälja vidare er biljett via Billio. Kontakta oss för att veta mer.

Coronaviruset (Covid-19) - Allmän information

Vänligen håll dig uppdaterad om status och rekommendationer från Folkhälsomyndigheten.

För de flesta evenemang krävs det nu Covidbevis för att komma in på evenemanget. Om du inte har beviset kan du inte komma in på evenemanget och dessutom inte få pengarna tillbaka.

Om ett evenemang behöver ställas in kommer ni få information direkt från arrangören om eventuellt återköp.
